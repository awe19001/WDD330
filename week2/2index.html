    const - used when the variable will not be reassigned to another value

    let - used if the variable might be reassigned later in the program

    Using const and let to declare variables means they are block scoped, their value only exists inside the block they are declared in.

    Any variable declared outside of a block is said to have global scope - meaning accessible everywhere in the program.

    Any variables inside a block using the let or const will only be available inside that block and not be accessible outside of that block is known local scope.

    If let or const are not used, the variable will have global scope and be available outside the block

    Constant and variable names can start with any upper or lower-case letter, an underscore, _, or dollar character, $. They are case sensitive too.

    String - a collection of characters. We can create a string literal by writing characters inside quote marks like this 'hello'.

    If you want to use an apostrophe in your string, you will need to use double quote marks to enclose the string, otherwise the apostrophe will terminate the string, causing an error.

    The backslash is used to escape special characters in strings such as: Single quote marks \' Double quote marks \" End of line \n Carriage return \r Tab \t

    Properties are information about the object or value, while methods perform an action on the object or value ― either to change it or to tell us something about it. To call a method, we use the dot operator (.) followed by the name of the method, followed by parentheses (this is a useful way to distinguish between a property and a method ― methods end with parentheses).

    A shortcut for string concatenation is to use the + operator to add the two strings together: 'Java' + 'Script' + ' ' + 'Ninja'; << 'JavaScript Ninja'

    NaN is an error value that is short for "Not a Number"

    Null means “no value”.

    Placing the ! operator in front of a value will convert it to a Boolean and return the opposite value. So truthy values will return false, and falsy values will return true. This is known as negation.
